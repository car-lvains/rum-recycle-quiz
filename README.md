# RUM 資源循環クイズ

NPO法人RUMアライアンス向けの子ども向けインタラクティブ学習Webアプリ
「クルマの冒険、終わらない旅 〜100%リサイクルに挑む最新技術〜」のソースです。

## 構成

- `index.html` ... 単一ファイルで完結する静的Webアプリ（Tailwind CSS / Chart.js / Web Audio API）
- iPad横向き表示に最適化

## ローカル確認

`index.html` をダブルクリックしてブラウザで開くだけで動作します。

## デプロイ

GitHub に push すると Vercel が自動でデプロイします。
ビルド設定は不要（静的ファイルのみ）。

- Framework Preset: **Other**
- Build Command: （空欄）
- Output Directory: `./`
